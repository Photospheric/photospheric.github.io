Based on Panellum,
for sharing panoramic images quickly

Photospheric is a tool I created to provide quick access to my panoramic images. You can also provide a quick way to view your panoramas through a special URL.

Quickstart

Step 1

Firstly, you need to host a panoramic image as well as an information XML file.
The XML file structure looks something like this: 
                    <PanoramaInfo>
    <PanoramaURL url="PANORAMIC IMAGE URL"/>
    <Title>PANORAMA TITLE HERE</Title>
    <Author>AUTHOR NAME HERE</Author>
    <InfoPageHTML>
        THIS TAG CONTAINS HTML SHOWN IN THE "image info" TAB.
        IT CAN CONTAIN ANY TYPE OF TAGS OR TEXT.
    </InfoPageHTML>
</PanoramaInfo>
XML code example
This example is hosted on:
https://raw.githubusercontent.com/Andryup/my-panoramas/refs/heads/main/data/1.xml 
                        <PanoramaInfo>
    <PanoramaURL url="https://raw.githubusercontent.com/Andryup/my-panoramas/refs/heads/main/data/images/rosendal_plains_2_4k.jpg"/>
    <Title>Rosendal Plains 2</Title>
    <Author>Dimitrios Savva & Jarod Guest</Author>
    <InfoPageHTML>
        Panoramic image downloaded from <a href="https://polyhaven.com/">Poly Haven</a>:
        <br>
        https://polyhaven.com/a/rosendal_plains_2
        <h3>Panoramic image location:</h3><br/>

    </InfoPageHTML>
</PanoramaInfo>

Step 2

Now we can finally move on to making a Photospheric URL that can be shared:
https://photospheric.github.io/view.html#xrf=<XML FILE URL HERE>
For example if our XML URL is:
https://raw.githubusercontent.com/Andryup/my-panoramas/refs/heads/main/data/1.xml

Then Photospheric URL will be: 
https://photospheric.github.io/view.html#xrf=https://raw.githubusercontent.com/Andryup/my-panoramas/refs/heads/main/data/1.xml
You can also encode it with Base64 if you wish to:
